## Installation via CDN

### CSS
```html
<!-- emojiRain css -->
<link href="https://iamovi.github.io/emojiRain/dist/emojirain.css" rel="stylesheet">
```

### JS
```html
<!-- emojiRain JS -->
<script src="https://iamovi.github.io/emojiRain/dist/emojirain.js"></script>
```
